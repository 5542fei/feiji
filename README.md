# 🎮 飞机坦克大战游戏

一个用原生 HTML + JavaScript 开发的经典射击小游戏，复刻了童年经典的飞机大战玩法！

---

## 📝 项目介绍
这是一个基于浏览器的 2D 射击游戏，玩家可以操控飞机移动、发射子弹，击毁敌方坦克/飞机获得分数，挑战最高分记录！

- 纯前端实现，无需任何额外依赖
- 打开 `index.html` 就能直接玩
- 支持键盘控制，操作简单易上手

---

## 🎯 游戏玩法
- **移动控制**：方向键 ← ↑ → ↓ 控制飞机移动
- **发射子弹**：自动发射 / 空格键发射（根据你的游戏逻辑调整）
- **游戏目标**：击毁敌方目标，获取更高分数，躲避敌方攻击

---

## 🚀 如何运行
### 方法 1：本地直接打开
1.  下载项目里的 `index.html` 和 `game.js` 文件
2.  双击 `index.html`，用浏览器打开就能直接玩！

### 方法 2：部署到线上
可以用 GitHub Pages 直接把游戏变成在线可玩的链接，步骤如下：
1.  进入仓库的 `Settings` → `Pages`
2.  选择 `Deploy from a branch`，分支选 `main`，文件夹选 `/root`
3.  保存后等待几分钟，就能获得一个在线链接啦！
